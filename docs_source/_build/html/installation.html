Installation¶

Stable release¶

To install FDR Hacking, run this command in your terminal:

$ pip install fdr_hacking

This is the preferred method to install FDR Hacking, as it will always install the most recent stable release.

If you don’t have pip installed, this Python installation guide can guide you through the process.

From sources¶

The sources for FDR Hacking can be downloaded from the Github repo.

You can either clone the public repository:

$ git clone git://github.com/kanduric/fdr_hacking

Or download the tarball:

$ curl -OJL https://github.com/kanduric/fdr_hacking/tarball/master

Once you have a copy of the source, you can install it with:

$ python setup.py install
